Let's Multiply!

Welcome to the multiplication micro-lesson. Multiplication is like adding a number to itself a certain number of times. For example, 3 multiplied by 4 (written as 3 x 4) is the same as adding 3 four times: 3 + 3 + 3 + 3 = 12.
